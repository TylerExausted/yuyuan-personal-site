# Yuyuan Chen Personal Website

A Vite + React personal website ready to deploy on Vercel.

## Local setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Vercel

1. Upload this folder to a GitHub repository.
2. Go to Vercel and choose **New Project**.
3. Import the GitHub repository.
4. Vercel should detect Vite automatically.
5. Click **Deploy**.

## Resume request form

The resume request form currently works as a front-end demo. To receive actual requests, connect it to Formspree, Google Forms, EmailJS, or a small backend API.
